Datatype Conversion

  • implicit
  • explicit

Implicit

will happen if there is no data loss, and no exceptions to be thrown. For example, from a smaller datatype to a larger.

Explicit

  • The Typecase Operator
    • if datatype cannot hold the variable amount, does not throw an exception
  • The Conversion Class
    • several methods for different datatypes
    • if datatype cannot hold the variable amount, throws an exception
  • Parse and Try Parse
    • Parse
    • can be used when converting from int to string types, and visa versa.
    • eg: int.Parse(variableName)
    • Try Parse
    • In case the original datatype cannot be converted to the target datatype.
    • eg: int.TryParse(variableName, out outputResultVariable)
    • TryParse as a method also takes on the values of either 'true' or 'false', depending on whether the parse is successful or not.
    • can use this in an 'if' statement.
    • use Parse if you are sure it will work.
